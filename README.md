# 🚀 HabitX - Smart Habit Tracker App

> 💡 A futuristic Android habit tracking app designed to improve consistency, productivity, and daily routines with smart features and interactive UI.

---

## 📱 Features

- 🔐 Login & Signup system  
- 🚀 Splash screen for smooth app launch  
- 🎨 Futuristic and modern UI design  
- 📊 Advanced stats with insights + chat support  
- 🏆 Achievement badges system  
- 📓 Personal journal for daily reflections  
- 👤 User profile management  
- 🔔 Real-time habit reminder notifications  
- 🔊 Sound effects for better user experience  
- 📤 Export habit reports  
- 🎯 Custom app icon  
- 📱 Fully tested on real Android device  

---

## 🛠 Tech Stack

- Kotlin (Android Development)
- HTML, CSS, JavaScript (WebView Integration)
- Android Studio
